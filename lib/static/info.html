Mumoro demonstration on the city of Rennes
Demonstration of our multi-modal and multi-objectives routing algorithms


Informations
  • This website is a demonstration of multi-modal and multi-objectives routing algorithms.
    Multi-modal routing combines multiple transport means such as car, public bikes and municipal transports (subway, bus).
    Multi-objectives routing results consider duration, price, number of line and/or modal changes.
    Routing is currently available only on the city of Rennes due to a lack of data on public transports on other French cities.
  • Data for public bikes Le vélo STAR are provided by Data Keolis Rennes
    Streets data and map visualization powered by Open Street Maps
  • Source is hosted and available at Git Hub
  • For bug reports and suggestions, please refer at Git Hub issues section, or contact developer Tristram Gräbener tristramg@gmail.com
  • Next releases will include:
    • Detailed routing page including information concerning directions and transports.
    • Possibility to send via a short URL a precise route to a friend, or print it to an image.
    • Public Transport of Rennes (as soon as the data is available)
Latest update : 20/04/2010

Informations
  • Ce site est une démonstration des algorithmes d'itinéraires multimodale et multiobjectifs.
    Un itinéraire multimodale combine tout moyen de transport tel que voiture, vélo publique et transport en commun (métro, bus).
    Les résultats multiobjectifs d'un itinéraires renvoient la durée du trajet, le cout et le nombre de changements de lignes et/ou moyens de transport.
    Les calculs d'itinéraires sont actuellement disponibles uniquement pour la ville de Rennes en raison de manque de données sur les transport en publiques sur d'autres villes de France.
  • Les données pour les vélos publiques Le vélo STAR sont fournis pas Data Keolis Rennes
    Les données routières ainsi que la visualisation de la carte sont apportes par Open Street Maps
  • Le code source du projet est hébergé et disponible chez Git Hub
  • Pour déclarer des bogues ou suggérer des améliorations reportez vous a la section des bogues de Git Hub, ou contactez le développeur Tristram Gräbener tristramg@gmail.com
  • Les prochaines mise à jour incluront :
    • Une feuille de route détaillant les directions et les moyens de transports.
    • Possibilités d'envoyer l'itinéraire via une URL courte a des amis, ou a générer une image.
    • Transports en commun de Rennes (dès que les données sont disponibles).
